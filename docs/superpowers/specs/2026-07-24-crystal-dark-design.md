# Crystal Dark — 个人网站视觉升级设计

> 日期: 2026-07-24
> 项目: 阿昕个人作品集网站
> 当前状态: 已批准

---

## 1. 设计目标

在保留现有「月之暗面」紫粉暗色基调的基础上，通过**玻璃质感、3D 深度、微交互、动态光效**四大维度进行视觉升级，让网站同时承载三个目标：
- **技术/求职** — 展示专业能力和细节品质
- **创意/个人品牌** — 传递独特审美和风格记忆点
- **产品变现** — 增强 Weekly OS 的产品质感和转化力

---

## 2. 色彩体系

保持暗色基底，新增「结晶质感」色彩层：

| 角色 | 色值 | 用途 |
|------|------|------|
| 🌑 基底 | `#05060a` | 页面背景（比纯黑更柔和） |
| 💜 主色 | `#a855f7` → `#c084fc` | 按钮、链接、强调 |
| 🌸 辅色 | `#ec4899` | 次强调、hover 状态 |
| 💎 结晶高光 | `#8a6bff` → `#4fc3ff` | 第二渐变、水晶光效、特殊卡片 |
| ✨ 强调色 | `#fbbf24` | 金色高亮、星标、价格等关键元素 |
| 🪟 玻璃面 | `rgba(255,255,255,0.03~0.08)` | 卡片/面板背景 |
| 🔲 玻璃边框 | `rgba(255,255,255,0.06~0.12)` | 卡片边框 |

## 3. 玻璃质感系统 (Liquid Glass)

所有卡片全面升级为多层玻璃效果：

```css
.glass-card {
  background: linear-gradient(
    135deg,
    rgba(255,255,255,0.06) 0%,
    rgba(255,255,255,0.02) 100%
  );
  backdrop-filter: blur(20px) saturate(1.4);
  border: 1px solid rgba(255,255,255,0.08);
  box-shadow:
    inset 0 1px 0 rgba(255,255,255,0.1),
    0 0 40px rgba(168,85,247,0.06);
}
```

## 5. Hero 区

### 布局
```
┌─────────────────────────────────────┐
│          ✦ 浮动水晶粒子 ✦            │
│     ◇   ●   ◇   ●   ◇   ●   ◇      │  ← 3D 浮动几何体
│        [● 在线 · 全栈开发者]          │
│          你好，我是 阿昕              │
│      用技术创造价值，以代码构建美好      │
│      [✨ 查看作品] [💬 联系我]        │
│            ▼ 向下滚动                 │
└─────────────────────────────────────┘
```

### 新增特效
1. **3D 浮动晶体** — Canvas 上绘制旋转多面体（八面体/十二面体），半透明紫/青色，鼠标拖拽时视差旋转
2. **鼠标光照** — 光晕跟随鼠标位置偏移，模拟手电筒光照
3. **磁性按钮** — 鼠标靠近时按钮向鼠标方向偏移 + 光晕跟随，离开时 spring 回弹
4. **滚动视差** — Hero 内容以不同速度消失，产生纵深层次感

## 6. 微交互系统

| 元素 | 交互效果 | 实现方式 |
|------|---------|---------|
| 导航链接 | 下划线从中间展开 + 文字微上浮 | CSS transition + transform |
| 按钮 | 磁性跟随 + 点击涟漪 + spring 回弹 | JS mousemove + CSS animation |
| 卡片 | 3D 倾斜（跟随鼠标角度）+ 边缘发光 | JS → CSS perspective/transform |
| 技能条 | 进入视口时数字跳动计数 (0→90%) | IntersectionObserver + setInterval |
| 时间线 | 节点脉冲显现 + 连线流动光点 | IntersectionObserver + CSS animation |
| 表单输入 | 聚焦时底部渐变线流动动画 | CSS focus-within + gradient |
| 滚动进度 | 顶部细进度条 (0%→100%) + 当前菜单高亮 | JS scroll + IntersectionObserver |
| 链接悬停 | 下划线彩虹渐变流动 | CSS animation + gradient |
| 卡片列表 | 进入视口 stagger 延迟 (0.08s/项) | IntersectionObserver + delay |

### 滚动动画参数
- 进入: `opacity: 0 → 1`, `translateY(40px) → 0`, 0.6s spring easing
- stagger 延迟: 每子项 0.08s
- 退出: `opacity → 0`, `scale → 0.95`
- 技能条: 进入后延迟 0.3s 才触发（制造节奏感）

## 12. 响应式策略

| 断点 | 行为 |
|------|------|
| >768px | 3D tilt、视差、粒子全部启用 |
| 480-768px | 粒子密度降50%，关闭 3D tilt（降为普通 hover） |
| <480px | 关闭粒子 Canvas，关闭视差，保持基础玻璃效果 |
