📷 项目截图预览

项目简介

一个实时数据可视化仪表盘，帮助企业监控关键业务指标，支持自定义图表和智能预警。

核心功能

  • 实时数据流展示（WebSocket）
  • 多种图表类型（折线图、柱状图、饼图等）
  • 自定义仪表盘布局
  • 数据导出为 Excel / PDF
  • 异常数据智能预警通知

技术亮点

前端使用 Vue.js 3 + D3.js 实现数据可视化，后端通过 WebSocket 推送实时数据。 支持拖拽式布局，用户可自由组合图表模块，满足不同场景的分析需求。
