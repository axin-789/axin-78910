电商平台

一个全功能的在线购物系统

React Node.js MongoDB
📷 项目截图预览

项目简介

这是一个全功能的电商平台，用户可以在上面浏览商品、加入购物车、下单购买。后台管理系统支持商品上架、订单处理、数据统计等功能。

核心功能

  • 用户注册 / 登录（JWT 认证）
  • 商品浏览、搜索、分类筛选
  • 购物车管理与下单流程
  • 在线支付集成（支付宝 / 微信）
  • 订单追踪与历史记录
  • 后台管理系统（商品、订单、用户管理）

技术亮点

前端使用 React + TypeScript 构建，采用组件化开发模式。后端基于 Node.js + Express，使用 MongoDB 存储数据。通过 Redis 缓存热点数据，首屏加载性能提升 40%。
